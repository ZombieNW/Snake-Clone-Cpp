## What?

One of my favorite ways to get familiar with a new language is by remaking classic arcade games! I am posting this for accomplishment and reference reasons.

## Building

Install g++ and make and all that good stuff.
Install [Raylib](https://www.raylib.com/).

```sh
make
```
